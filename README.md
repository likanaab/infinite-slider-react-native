Реализация замкнутой карусели из баннеров с автопрокрутом в 3.5 секунд.
Автопрокрутка автоматически останавливается при взаимодействии пользователя со слайдерами, а затем возобновляется по окончанию жеста.

Вариант без использования готового метода слайдера

Использован react-native-reanimated
